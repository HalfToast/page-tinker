# Privacy Policy

**Page Tinker** does not collect, store, or transmit any personal information.

## Data sent to third parties

None. The extension makes no network requests of any kind. Nothing about the
page you are on, the changes you make, your browsing history, URLs, identifiers,
or any other data ever leaves your device.

## What is stored locally

Nothing. The extension uses no storage APIs. Every change you make (edit, hide,
delete) is held only in the live page and is discarded the moment you reload or
navigate away.

## Page access

Page Tinker has no host permissions. It can only touch a page after you
explicitly invoke it - via the toolbar button, the right-click menu item
"Start Page Tinker on this page", or the `Alt+Shift+E` shortcut - using the
`activeTab` permission, and only the tab you invoked it on. It never runs
automatically or in the background. The `contextMenus` permission only adds
that one menu entry and grants no access to page content.
